# Luceat

Luceat is a finite idle fantasy RPG about growing from a lone aspirant into a guild adventurer.

The first playable slice centers on personal progression:

- Earn Gold, Martial EXP, and Magic EXP.
- Build Resolve Reserve instead of spending energy.
- Attempt readiness-gated quests when your character can endure them.
- Earn admission to the Adventurer's Guild without time-based objectives.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
