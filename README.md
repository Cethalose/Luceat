# Luceat

Luceat is a locally hosted AI-powered tabletop RPG Game Master.

## Vision

Rather than relying on an AI model to remember an entire campaign, Luceat acts as the game engine while a local language model provides immersive narration.

Responsibilities are divided as follows:

### Luceat
- Campaign management
- World state
- NPC tracking
- Quest tracking
- Story progression
- Player management
- Game mechanics

### AI (Gemma via Ollama)
- Scene narration
- NPC dialogue
- Descriptive storytelling
- Creative prose

## Current Features

- Local Ollama integration
- Discord bot
- Configurable Game Master prompt
- World data loading
- Channel restriction
- Modular architecture

## Future Goals

- Multiple campaigns
- Persistent NPC memory
- Quest system
- Story arc manager
- Desktop control panel
- World editor
- Dice and combat system
